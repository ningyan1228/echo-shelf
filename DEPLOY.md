# Deploy 拾音匣 To GitHub Pages

Upload the contents of this folder to the root of a new GitHub repository.

Required files:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `LICENSE`
- `robots.txt`
- `.nojekyll`
- `assets/default-cover.svg`
- `assets/preview.png`

GitHub Pages settings:

1. Open repository `Settings`.
2. Open `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Choose branch `main`.
5. Choose folder `/root`.
6. Save.

Do not upload `MoeKoeMusic-main`, `node_modules`, `api`, build output from unrelated projects, or any third-party music platform source code.
